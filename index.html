Sobre o Cleyton

Minha jornada na área de desenvolvimento de sistemas começou em 2015, quando me aventurei na programação web com PHP. Rapidamente, ingressei na faculdade de Ciência da Computação, onde fui introduzido às linguagens C, C++, e C#. A medida que construía diversos projetos nessas linguagens, desenvolvi um apreço especial por programação robusta, cultivando uma paixão por Java. Após um ano, decidi explorar o mundo dos aplicativos móveis, optando por desenvolver em Android utilizando tanto Java quanto Kotlin. Embora tenha enfrentado alguns desafios devido ao meu conhecimento limitado em Java na época, obtive uma sólida experiência após a conclusão de dois aplicativos. Para aprimorar minhas habilidades, busquei e obtive certificações em Java e SQL Server em 2019. Com um conhecimento abrangente em desenvolvimento web, mobile e sistemas, estou confiante em oferecer meus serviços para auxiliar empresas e empreendedores individuais a alcançarem seus objetivos. Estou entusiasmado para aplicar minha experiência e expertise para agregar valor aos projetos que me forem confiados.

Portifólio Oportunidades Desenvolvimento

Criação de Web Services

Com base inicial de ferramenta de gestão de identidade da SAP, o cliente procura soluções de integrações com sistemas non-sap para alimentar de forma síncrona todos os sistemas da empresa. A criação de Web Services foi uma das formas mais utilizadas por mim em diversos projetos, sendo eles os clientes (Companhia Siderurgia Nacional, MRS Logística, ATVOS) entre outros. Para o Web Service trabalhar de forma síncrona, foi realizado o desenvolvimento na linguagem Java para consumo de API REST disponibilizada pelo cliente e Javascript para consumo de API SOAP (Este método direto na ferramenta de gestão de identidade da SAP). Com isso, foi utilizado como formas de autenticação oAuth2 para por exemplo (Docusign) e criptografia TLS utilizado no ambiente (PeopleSoft). Além destes exemplos, foi utilizado o desenvolvimento do Web Service como conectores customizados de sistemas non-sap. Abaixo um modelo de Web Service Java e Javascript utilizado: 

Java: https://github.com/CleytonPrudencio/sobre/blob/gh-pages/WebServiceJava 
Javascript: https://github.com/CleytonPrudencio/sobre/blob/gh-pages/WebServiceJavascript 

Criação de API

Criação de API utilizando Swagger dentro de um sistema formado por camada MVC utilizando diversos frameworks como Springboot, struts entre outros. Neste sistema, iniciei a criação de métodos dentro da API com base nas funcionalidades que o sistema já possuía, sendo assim, trabalhando com o código da aplicação onde muitas vezes possuía apenas a controller e dados. Em outras oportunidades tive o prazer de criar a estrutura completa com métodos diferentes do que usados na interface com usuário, sendo assim gerando um outro controller para a API tratar exceções, erros e segurança através da view da API. Além disto, adicionando como cabeçalho a Token como forma de conexão. Como questão de segurança, não e possível compartilhar o código ou mostra-lo, porem a API e publica e está disponível no link: 
https://api.totaldocs.com/v1/ 
https://farm-agentes-cobranca.dev-api.portalfarm.com.br/swagger-ui/index.html

Criação de Sistemas Web

Para criação de Sistemas Web, um dos maiores projetos foi onde obtive conhecimento do framework struts, que trabalha de forma logica as condições e tratativas das páginas JSP e do que elas podem ter acesso ou não dependendo de permissão das classes controller de usuários. Além disto, a criação de métodos e configuração das classes e métodos criados dentro do struts e passando isso para a JSP foi um caminho complexo e com bastante aprendizado. Além disto, aplicado também o conhecimento de Javascript, ajax (para chamar o método na JSP e tratar resposta) HTML, CSS e Bootstrap para tratar layout. Conexoes com o banco de dados obtive em todos os casos de Sistema, API e ate mesmo Web Service, sendo assim, uma atividade simples. Como questão de segurança, não e possível compartilhar o código ou mostra-lo. Porem separei um exemplo de sistema criado a mão para compartilhar conhecimento. https://github.com/CleytonPrudencio/sobre/tree/gh-pages/CatalogBrinquedo
